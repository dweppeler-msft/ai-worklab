# AI Worklab

Interaktives Mini-Lab zum Thema **KI im Arbeitsalltag**.

**Live:** https://dweppeler-msft.github.io/ai-worklab/

Fünf Berufe, je drei Aufgaben aus echten Arbeitssituationen. Du entscheidest, wie KI klug eingesetzt wird – und siehst dann, wie es die Profis machen.
